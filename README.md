# TechNova 🚀
- Proyecto de comercio electrónico
## Descripción
TechNova es un sistema de comercio electrónico orientado a la venta de productos tecnológicos.

## Funcionalidades
- Registro de usuarios
- Catálogo de productos
- Carrito de compras
- Gestión de pedidos

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- MySQL
